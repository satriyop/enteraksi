# Course Features
## Course Settings:
- course visibility : public, restricted, hidden.
- course enrollment eligibility : require to login and approval by lms admin(private)

- course status : draft, published, archieved
- self-pacing and scheduled course
- certificate eligibility criteria
- course completion requirements
- grade calculation method
- late submission policy
- attendance tracking enabled/disabled

# Course Structure
## Organizational Units:
- Curriculums (Managed at the institutional or admin level, often through "Program Management" features)
- Learning path (enrollment into a series of courses and tracks overall progress through that specific journey, often with a visual progress bar.)
- Syllabus / Course (includes a general overview, a link to the grading policy, a calendar with due dates, and links to the following components : Milestones, Chapters, Topics, Lessons)
- Milestones
- Chapters (focused collection of related learning activities and content within a milestones)
- Topics
- Lessons (actual content delivered to the student; the core learning material.)
## course content management
- authoring content : WYSIWYG editor for text content, upload assets, drag-and-drop module order.
- content components: module, lesson (video, PDF, quiz, youtube).
- trainer, content manager, lms admin able to create, edit, delete assignment and quizzes.
- course/content approval.


## course lesson content types:
- Video lectures
- Reading materials (PDF, DOC, presentations)
- Interactive content (H5P, SCORM)
- External links
- Embedded content (YouTube, Vimeo)
- Live sessions (integration with Zoom/Google Meet/Youtube live)
- Downloadable resources
- Assignments
- Quizzes and assessments
- Discussion forums
- Surveys

## Learning Experience
- Tracks all learning experiences

## Course Enrollment
- user enrolled to particular course, will be logged extensively to collect informations such as : what courses are enrolled for specific user, what is the progress of each enrolled course(up to section, topic/sub topic).
- user taking lesson are being tracked extensively, so user able to resume the latest lesson taken by the user and to collect other useful information.
