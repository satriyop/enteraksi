# Assessments & quizzes
- Question types: multiple-choice, multi-answer, true/false, short answer, essay, file upload, simulation/case study.
- Timed quizzes, pass thresholds, attempts configuration.
- Automatic grading for objective types; manual grading workflow for essays with trainer comments.
- Grading scale & competency matrix mapping to job roles.
- Plagiarism detection integration (optional).
